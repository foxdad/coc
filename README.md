# COC 文字版 - 部落冲突文字游戏

> 🎮 **在线体验**: [https://coc.xiaokexian.top/](https://coc.xiaokexian.top/)

一款基于 Vue3 开发的文字版部落冲突游戏，还原经典 COC 玩法，纯文字交互体验。

## 游戏特色

- 完整的 1-9 本升级体系
- 资源采集与存储系统
- 兵种训练与研究升级
- 英雄系统（野蛮人之王、弓箭女皇）
- 哥布林副本挑战
- 进攻掠夺战斗系统
- 村庄树木挖掘（有几率获得宝石）
- 日间/夜间主题切换
- 存档导入导出（加密保护）

## 核心玩法

### 资源系统
- 金币、圣水、暗黑重油（7本解锁）
- 建造金矿、圣水收集器产出资源
- 储金罐、圣水瓶扩展存储上限

### 建筑系统
- 大本营：核心建筑，解锁更多内容
- 资源建筑：金矿、圣水收集器、暗黑钻井
- 军事建筑：兵营、暗黑兵营、实验室
- 防御建筑：加农炮、箭塔、迫击炮

### 兵种系统
- 普通兵种：野蛮人、弓箭手、巨人、哥布林、炸弹人、气球兵、法师、天使、飞龙、皮卡超人
- 暗黑兵种：亡灵、野猪骑士、熔岩猎犬（7本解锁）
- 实验室升级兵种等级

### 英雄系统
- 野蛮人之王：7本解锁，最高20级
- 弓箭女皇：9本解锁，最高10级

### 战斗系统
- 搜索对手消耗金币
- 根据兵种配置模拟战斗
- 获得金币、圣水、奖杯奖励
- 战斗会消耗所有部队

### 哥布林副本
- 单人PVE关卡挑战
- 通关获得大量资源奖励
- 每日挑战次数限制

## 技术栈

- Vue 3 + Composition API
- Pinia 状态管理
- Vite 构建工具
- 纯 CSS 样式（无UI框架）

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 游戏设置

- 资源产出倍率：1-10倍可调
- 主题模式：日间/夜间切换
- 数据管理：导出/导入加密存档
