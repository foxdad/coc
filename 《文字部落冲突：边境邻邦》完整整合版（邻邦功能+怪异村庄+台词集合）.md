# 《文字部落冲突：边境邻邦》完整整合系统设计文档

## 一、 设计总纲（核心逻辑统一）

所有邻邦（原有 3 种 + 新增 3 种怪异村庄）遵循同一套核心规则：



1. 以「好感度阶梯」（-100\~+100）为基础，决定互动权限、剧情触发、收益惩罚；

2. 兵种赠予 / 回赠、冲突、剧情互动均围绕「邻邦关系」展开，新村庄与原村庄共享同一套触发逻辑；

3. 怪异村庄的「性感怪异」风格通过台词、视觉设定体现，但核心数值（好感度变化、回赠强度、攻城规模）与原有村庄保持平衡一致。
